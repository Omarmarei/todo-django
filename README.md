# ToDo-list-App

## Technologies
* Python
* Django
* HTML
* CSS
* Markdown conversion

## Summary
Welcome to our project! This is a Todo list App made in Python and the Django framework. It allows users to quickly and easily add, delete, and update events and tasks. With this App, users can effortlessly manage their tasks and events, helping them stay organized and on top of their to-do list

## How to Run
1. Clone this repository, navigate to the project and type the following commands:
2. Activate a virtual environment: 'python3 -m venv .venv' and select the virtual environment as the active workspace
3. Install dependencies: 'pip install -r requirements.txt'
4. Run the project: 'python3 manage.py runserver'
